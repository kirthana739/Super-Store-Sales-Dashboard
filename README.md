# Superstore Sales Dashboard

## About
Analyzed 9000+ row Kaggle Superstore dataset using Microsoft Excel.

## What I Did
- Data Cleaning (fixed date formats, hidden empty columns, 
  highlighted 1871 loss-making orders)
- Documented all changes in Change Log sheet
- Data Summarization using Pivot Tables
- Used SUMIF, SUMIFS, SUMPRODUCT, VLOOKUP formulas
- Built 4-chart interactive dashboard

## Dashboard Charts
- Total Sales by Region
- Profit by Category
- Loss Analysis by Category
- Monthly Sales Trend

## Key Findings
- West region has highest sales (725,457)
- Technology is most profitable (145,454)
- Office Supplies has highest loss (-60,936)
- Sales peak in November-December
- Furniture has lowest profit (18,451)
